# Deployment Model


### Current Deployment

**Development**:
```bash
npm run dev
```
- Port 5173 (Vite dev server)
- Port 3001 (API server)

**Production Build**:
```bash
npm run build
npm run preview
```
- Static file generation
- Optimized bundles
- Preview server

### Requirements

**System**:
- Node.js 18+
- npm 9+
- Modern browser

**Optional**:
- Docker (for backend services)
- Git (for version control)
