# Current Limitations


### Technical Constraints

1. **Storage**:
   - Limited by browser localStorage (~5-10MB)
   - No server-side persistence
   - Single-user only

2. **Collaboration**:
   - No real-time multi-user
   - No conflict resolution
   - No change history

3. **Export**:
   - Markdown only
   - No PDF generation
   - No image export of canvas

4. **Performance**:
   - Client-side rendering only
   - No virtualization
   - Large datasets may slow down

5. **Browser**:
   - Requires modern browser
   - No IE support
   - Must enable JavaScript

### Functional Limitations

1. **Ideation**:
   - No collaboration features
   - No version control
   - No templates library

2. **Storyboard**:
   - No auto-layout
   - No swimlanes
   - No time estimation

3. **Integration**:
   - Limited external tool support
   - No API for external access
   - No webhooks
