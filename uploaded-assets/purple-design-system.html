Design Principles

1. Material Design Foundation

This system is built on Material Design principles, emphasizing:

  • Clear hierarchy through elevation and shadows
  • Meaningful motion and transitions
  • Consistent spacing and proportions
  • Bold, intentional use of color

2. Accessibility First

  • All color combinations meet WCAG AA contrast requirements
  • Focus states are clearly visible
  • Touch targets are minimum 44x44px
  • Semantic HTML is used throughout

3. Purple Identity

The purple color palette differentiates this system while maintaining Material Design's clarity:

  • Primary purple (#7b1fa2) for key actions and branding
  • Lighter purples for backgrounds and subtle accents
  • Semantic colors for feedback (success, warning, error)
  • Neutral grays for text and supporting elements

4. Responsive & Flexible

  • Mobile-first approach
  • Flexible grid system
  • Scalable typography
  • Adaptive components

Implementation Guide

Getting Started

To use this design system in your project:

  1. Include the Roboto font from Google Fonts: 
    <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@300;400;500;700&display=swap" rel="stylesheet">
  2. Copy the CSS from this document into your stylesheet
  3. Use the provided HTML structure and classes
  4. Customize colors using CSS variables if needed

Best Practices

  • Use the elevation system to show hierarchy
  • Maintain consistent spacing using the spacing scale
  • Follow the typography scale for text sizing
  • Test color contrast for accessibility
  • Use semantic HTML elements
  • Ensure keyboard navigation works properly

Customization

To customize colors, update these key values:

/* Primary Colors */
--primary: #7b1fa2;
--primary-dark: #6a1b9a;
--primary-light: #8e24aa;

/* Secondary Colors */
--secondary: #9c27b0;
--secondary-light: #ab47bc;
