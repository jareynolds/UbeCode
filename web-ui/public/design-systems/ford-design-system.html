Spacing System

Consistent spacing creates visual rhythm. Our system uses an 8px base unit for predictable, harmonious layouts.

Spacing Scale

xs (8px) - Minimal spacing between closely related items
sm (16px) - Small spacing for compact layouts and form elements
md (24px) - Medium spacing for standard content separation
lg (32px) - Large spacing for distinct content sections
xl (48px) - Extra large spacing for major page sections
2xl (64px) - Maximum spacing for clear visual breaks

Ford Design Principles

1. Built Ford Tough Meets Modern Design

This design system honors Ford's heritage of strength and reliability while embracing contemporary digital design principles:

  • Bold, confident use of Ford's signature blue palette
  • Clear visual hierarchy through elevation and typography
  • Robust, functional components that perform reliably
  • Professional aesthetic that inspires trust

2. Go Further Philosophy

Every design decision supports Ford's "Go Further" mission:

  • Innovation: Modern, forward-thinking design patterns
  • Accessibility: Inclusive design for all users
  • Performance: Optimized for speed and efficiency
  • Sustainability: Thoughtful use of resources and energy

3. Brand Consistency

The official Ford Motor Company color palette ensures brand recognition:

  • Maastricht Blue (#081534): Primary brand color for headers and key elements
  • Dark Cerulean (#133A7C): Main interactive elements and CTAs
  • Lapis Lazuli (#2A6BAC): Secondary actions and links
  • Picton Blue (#47A8E5): Accents and highlights
  • Silver Sand (#C6C6C6): Borders and subtle UI elements

4. User-Centered Experience

  • Intuitive navigation and clear information architecture
  • Responsive design that works across all devices
  • Accessible components meeting WCAG AA standards
  • Fast loading times and smooth interactions

5. Material Design Foundation

Built on Material Design principles while maintaining Ford's unique identity:

  • Elevation system for depth and hierarchy
  • Consistent spacing and proportions
  • Smooth animations and transitions
  • Touch-friendly interaction targets

Implementation Guide

Getting Started

To implement the Ford Design System in your project:

  1. Include Roboto Font: 
    <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@300;400;500;700&display=swap" rel="stylesheet">
  2. Add the Design System CSS: Copy the CSS from this document into your stylesheet or include it as a separate file
  3. Use Component Classes: Apply the provided classes to your HTML elements
  4. Maintain Brand Colors: Always use the official Ford color palette
  5. Test Accessibility: Ensure all interactions are keyboard accessible and meet contrast requirements

CSS Variable Implementation

Define Ford brand colors as CSS variables for easy customization:

:root {
    /* Primary Ford Blues */
    --ford-maastricht-blue: #081534;
    --ford-dark-cerulean: #133A7C;
    --ford-lapis-lazuli: #2A6BAC;
    --ford-picton-blue: #47A8E5;
    
    /* Neutrals */
    --ford-white: #FFFFFF;
    --ford-silver-sand: #C6C6C6;
    --ford-slogan-gray: #3E5966;
    
    /* Extended */
    --ford-light-blue-tint: #E8F4FB;
    --ford-extra-light-blue: #F5FAFD;
    
    /* Spacing */
    --spacing-xs: 8px;
    --spacing-sm: 16px;
    --spacing-md: 24px;
    --spacing-lg: 32px;
    --spacing-xl: 48px;
    --spacing-2xl: 64px;
}

Best Practices

  • Color Usage: Use Maastricht Blue (#081534) for primary actions, Dark Cerulean (#133A7C) for secondary, and Picton Blue (#47A8E5) for accents
  • Typography: Maintain the type scale for consistent hierarchy
  • Spacing: Use the 8px-based spacing system for all margins and padding
  • Elevation: Apply appropriate shadow levels to show element hierarchy
  • Accessibility: Ensure minimum 44x44px touch targets and WCAG AA contrast ratios
  • Performance: Optimize images and minimize unnecessary animations
  • Testing: Test on multiple devices and browsers
