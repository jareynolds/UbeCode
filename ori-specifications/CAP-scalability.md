# Scalability


### Current Scale

**Single User**:
- 10-50 workspaces
- 100-500 ideation blocks per workspace
- 20-100 storyboard cards per workspace
- 50-200 tags per workspace

**Performance**: Acceptable for intended use

### Scale Limitations

**Breaking Points**:
- >500 blocks on ideation canvas
- >100 storyboard cards
- >1000 tags
- >10MB localStorage

**Mitigation**: Not currently implemented
