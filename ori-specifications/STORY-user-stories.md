# User Stories

**Status:** pending


### Primary User Stories

**US-1: As a product manager, I want to link storyboard cards to ideation concepts so that I can maintain context**
- Given I have tagged items in my ideation canvas
- When I create or edit a storyboard card
- Then I can select which ideation tags apply to this card

**US-2: As a developer, I want to see which ideation concepts relate to a story card so that I understand the "why"**
- Given a storyboard card has ideation tags
- When I view the card on the canvas
- Then I see the associated tags displayed as badges

**US-3: As a team lead, I want to export specifications that include ideation references so that I can share complete context**
- Given I export storyboard to markdown
- When the export completes
- Then ideation tags are included in the documentation
