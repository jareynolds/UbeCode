# Extensibility


### Extension Points

1. **New Page Types**:
   - Add to `src/pages/`
   - Register route in App.tsx
   - Add to navigation

2. **Custom Components**:
   - Create in `src/components/`
   - Import and use
   - Style with Apple HIG

3. **Context Extensions**:
   - Extend workspace interface
   - Add new context providers
   - Maintain backward compatibility

4. **Export Formats**:
   - Extend export functions
   - Add new endpoints
   - Support new file types
