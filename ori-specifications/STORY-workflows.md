### Tagging Workflow

1. **Ideation Phase**
   - Create text/image blocks in Ideation
   - Add descriptive tags (e.g., "user-flow", "mvp")
   - Tags saved to workspace

2. **Planning Phase**
   - Navigate to Storyboard
   - Create new story card
   - Open tag selection UI
   - Select relevant ideation tags
   - Tags appear on card

3. **Review Phase**
   - View storyboard
   - See tags on cards
   - Understand ideation context
   - Export to markdown
